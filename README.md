# uniqcast_task

**Getting started**  
  
1. Install flutter  
2. Make sure you have 'stable' version selected. `flutter --version`  (use Flutter version 3.19.5)
3. Open project.  
3. Run `flutter clean && flutter pub get`  
4. Connect mobile phone and run application
